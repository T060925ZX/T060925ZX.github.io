在Linux部署云崽(1)

2024/7/20 Yunzai 代码 Node Git

本文是手搓教程，不是一键脚本

环境准备： 一台Linux服务器，推荐使用Debian和Ubuntu系统（按顺序推荐）

换源

bash <(curl -sSL https://linuxmirrors.cn/main.sh)
  • 选择清华源

  • 确保稳定，按Y使用http

  • 一直点回车

  • 按n取消跳过

  • 清理一下

环境配置

所需环境 Nodejs Git Redis ffmpeg(发送语言和视频) chromium

先配置环境安装环境，有点服务器没有

apt update

apt install sudo

sudo apt install curl

使用nvm安装Nodejs

安装NVM包管理器

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash

刷新一下

source ~/.bashrc

安装Nodejs22

nvm install v22.3.0

检测是否安装成功

node -v # should print `v22.3.0`

npm -v # should print `10.8.1`

使用安装包安装Nodejs ( 不推荐 )

  • 安装node

cd 到/usr/local/目录

cd /usr/local/

新建一个nodejs的目录

mkdir nodejs

// 下载node

wget https://nodejs.org/dist/v22.5.1/node-v22.5.1-linux-x64.tar.xz

// 解压node

tar xf node-v22.5.1-linux-x64.tar.xz

// 重命名

mv node-v22.5.1-linux-x64 nodejs22

// 设置node软链接

ln -s /usr/local/nodejs/nodejs22/bin/node /usr/bin/node

检查node是否安装成功

node -v
  • 安装npm

// 配置npm软链接

ln -s /usr/local/nodejs/nodejs22/bin/npm /usr/bin/npm

//检查npm是否配置成功: 

npm -v
  • 安装pm2
npm install -g pm2

// 配置pm2软链接

ln -s /usr/local/nodejs/nodejs22/bin/pm2 /usr/bin/pm2

需要其他版本去nodejs官网找需要的版本

安装Git

sudo apt update

sudo apt install git

检测是否安装成功

git -v # should print `git version 2.39.2

安装Redis

sudo apt install redis

安装ffmpeg

sudo apt install ffmpeg

安装chromium

sudo apt install chromium-browser

现在你已经配置完成环境了，点击这里跳转下一步
