在Linux部署云崽(2)

2024/7/20 Yunzai 代码 Node Git

本文是手搓教程，不是一键脚本

环境准备： 一台配置好环境的Linux服务器，没有配置请点击这里

环境配置

环境要求：已经配置好 Nodejs Git Redis ffmpeg(发送语言和视频) chromium 的服务器

安装云崽本体

点击跳转你要的崽Miao-Yunzai TRSS-Yunzai

Miao-Yunzai

  • 克隆项目

请根据网络情况选择Github安装或Gitee安装

# 使用 Github 
git clone --depth=1 https://github.com/yoimiya-kokomi/Miao-Yunzai.git
cd Miao-Yunzai 
git clone --depth=1 https://github.com/yoimiya-kokomi/miao-plugin.git ./plugins/miao-plugin/

# 使用Gitee
git clone --depth=1 https://gitee.com/yoimiya-kokomi/Miao-Yunzai.git
cd Miao-Yunzai 
git clone --depth=1 https://gitee.com/yoimiya-kokomi/miao-plugin.git ./plugins/miao-plugin/

  • 安装pnpm

已安装的可以跳过

npm install pnpm -g

  • 安装依赖

外网环境请修改的本地npm配置.npmrc

# 直接安装
pnpm install -P

  • 运行

首次运行按提示输入登录

npm run app

  • 登录

npm run login

  • 托管

npm run start

遇到登录错误（45）请点击这里配置签名API

TRSS-Yunzai

  • 克隆项目

请根据网络情况选择Github安装或Gitee安装

# 使用 Github 
git clone --depth 1 https://github.com/TimeRainStarSky/Yunzai
# 使用Gitee
git clone --depth 1 https://gitee.com/TimeRainStarSky/Yunzai

cd Yunzai

  • 安装pnpm和依赖

npm i -g pnpm
pnpm i

  • 前台启动

操作 命令
启动 node .
停止 node . stop

  • 使用 pm2 后台运行 (进程守护)

操作 命令
启动 pnpm start
停止 pnpm stop
日志 pnpm log

  • 前台启动开机自启 (需要安装 pm2 )

安装 PM2

npm install pm2 -g

pnpm start
pnpm pm2 save
pnpm pm2 startup

启动TRSS-Yunzai后需要在控制台输入

#安装genshin
#安装miao-plugin
#安装TRSS-Plugin
