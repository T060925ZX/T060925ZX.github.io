Yunzai问题解决方案 Qsign

环境准备： 部署好的Yunzai-Bot

本文所以内容均不允许以任何形式传入国内流量平台

如：哔哩哔哩等

修改配置文件

提示：要先修改设备文件以兼容高版本签名9.0.17+

方法一：（ 此方法仅适用于喵崽 ）

将下载的文件放入Miao-Yunzai\node_modules\icqq\lib\core\device.js替换掉原来的device.js

点击这里下载文件

由于格式限制，下载后将device.js.txt重命名为device.js
然后配置 bot.yaml 文件点我跳转

方法二：（ 使用一键脚本 ）

选择你的崽类型执行以下代码

  • tsab TRSS-Yunzai
tsab c 'cd $DIR/TRSS-Yunzai && curl -sSL QSign.icu/X | bash'
  • tsab Miao-Yunzai
tsab c 'cd $DIR/Miao-Yunzai && curl -sSL QSign.icu/X | bash'
  • bash
cd 崽的目录

curl -sSL Qsign.icu/ver |bash
  • 白狐脚本 Miao-Yunzai
cd /root/Miao-Yunzai && curl -sSL QSign.icu/X | bash
  • 白狐脚本 TRSS-Yunzai
cd /root/TRSS-Yunzai && curl -sSL QSign.icu/X | bash

脚本一键配置会自动添加签名版本？应该是这样的

Miao-Yunzai修改方法

打开Miao-Yunzai/config/config/bot.yaml

在文件的末尾我们可以看到如下

# 签名API地址(如:http://127.0.0.1:8080/sign?key=114514)
sign_api_addr:
# 传入的QQ版本(如:8.9.63、8.9.68)
ver:

按以下格式填入（可以直接复制粘贴我的）

# 签名API地址(如:http://127.0.0.1:8080/sign?key=114514)（填你要用的API）
sign_api_addr: https://hlhs-nb.cn/signed/?key=114514
# 传入的QQ版本(如:8.9.63、8.9.68)（可填可不填，选择你要的版本）
ver:

TRSS-Yunzai修改方法

打开TRSS-Yunzai/config/ICQQ.yaml

tips:
  - 欢迎使用 TRSS-Yunzai ICQQ Plugin ! 作者：时雨🌌星空
  - 参考：https://github.com/TimeRainStarSky/Yunzai-ICQQ-Plugin
permission: master
markdown:
  mode: false
  button: false
  callback: true
bot:
  sign_api_addr: https://hlhs-nb.cn/signed/?key=114514
token: []

按以下格式填入（可以直接复制粘贴我的,自己修改）

tips:
  - 欢迎使用 TRSS-Yunzai ICQQ Plugin ! 作者：时雨🌌星空
  - 参考：https://github.com/TimeRainStarSky/Yunzai-ICQQ-Plugin

markdown:
  mode: false
  button: false
  callback: true
